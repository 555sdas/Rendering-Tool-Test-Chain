# 前端操作文档

## 1. 环境要求

| 组件 | 最低版本 | 推荐版本 |
|------|---------|---------|
| Node.js | 18 | 20 LTS |
| npm | 9 | 10 |
| 浏览器 | Chrome 90+ | Chrome最新版 |

## 2. 快速启动

### 2.1 安装依赖

```bash
cd frontend

# 使用npm
npm install

# 或使用yarn
yarn install

# 或使用pnpm
pnpm install
```

### 2.2 开发模式启动

```bash
# 启动开发服务器
npm run dev

# 服务启动后访问
# http://localhost:5173
```

### 2.3 生产构建

```bash
# 构建生产版本
npm run build

# 构建输出目录: dist/
# 包含所有静态文件，可直接部署到Nginx/Apache
```

### 2.4 预览生产构建

```bash
# 使用Vite预览生产构建
npm run preview

# 或使用Docker
npm run build
docker build -t xr-test-frontend .
docker run -p 80:80 xr-test-frontend
```

## 4. 开发指南

### 4.1 添加新页面

```typescript
// 1. 创建页面组件 src/pages/NewPage/index.tsx
import React from 'react';

const NewPage: React.FC = () => {
  return (
    <div>
      <h1>新页面</h1>
    </div>
  );
};

export default NewPage;

// 2. 添加路由 src/routes.tsx
import NewPage from './pages/NewPage';

const routes = [
  // ... 其他路由
  {
    path: '/new-page',
    element: <NewPage />,
    auth: true,  // 是否需要登录
    title: '新页面'
  }
];
```

### 4.2 添加新API

```typescript
// src/api/newApi.ts
import { apiClient } from './client';

export interface NewData {
  id: number;
  name: string;
}

export const newApi = {
  getList: () => apiClient.get<NewData[]>('/api/v1/new-endpoint'),
  create: (data: Partial<NewData>) => apiClient.post('/api/v1/new-endpoint', data),
  update: (id: number, data: Partial<NewData>) => apiClient.put(`/api/v1/new-endpoint/${id}`, data),
  delete: (id: number) => apiClient.delete(`/api/v1/new-endpoint/${id}`)
};
```

### 4.3 使用状态管理

```typescript
// 在组件中使用Zustand状态
import { useAuthStore } from '../stores/authStore';

const MyComponent: React.FC = () => {
  const { user, isAuthenticated, logout } = useAuthStore();
  
  if (!isAuthenticated) {
    return <div>请先登录</div>;
  }
  
  return (
    <div>
      <p>欢迎, {user?.username}</p>
      <button onClick={logout}>退出</button>
    </div>
  );
};
```

## 7. 构建与部署

### 7.1 Docker部署

```bash
# 构建镜像
docker build -t xr-test-frontend .

# 运行容器
docker run -d \
  -p 80:80 \
  --name xr-frontend \
  xr-test-frontend

# 查看日志
docker logs -f xr-frontend
```

### 7.2 Nginx部署

```bash
# 1. 构建
npm run build

# 2. 复制到Nginx目录
cp -r dist/* /var/www/html/

# 3. 配置Nginx (参考 nginx.conf)
# 已包含SPA路由回退、Gzip压缩、安全头

# 4. 重启Nginx
sudo nginx -s reload
```

### 7.3 CDN部署

```bash
# 构建生产版本
npm run build

# 上传到CDN
# 例如阿里云OSS、腾讯云COS、AWS S3等
ossutil cp -r dist/ oss://your-bucket/
```

## 10. 常见问题

### Q1: 安装依赖失败
```bash
# 清除缓存后重试
npm cache clean --force
rm -rf node_modules package-lock.json
npm install

# 或使用代理
npm config set registry https://registry.npmmirror.com
npm install
```

### Q2: 开发服务器启动失败
```bash
# 检查端口占用
lsof -i :5173

# 使用其他端口
npm run dev -- --port 3000
```

### Q3: 构建失败
```bash
# 检查TypeScript错误
npm run typecheck

# 检查内存不足
NODE_OPTIONS="--max-old-space-size=4096" npm run build
```

### Q4: API请求跨域错误
```bash
# 检查后端CORS配置
# 检查vite.config.ts代理配置
# 检查环境变量VITE_API_BASE_URL
```
